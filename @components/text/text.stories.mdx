### Ellipsis

By using the prop `ellipsis` it is possible to limit the text to one line and to
signal to the user there is more to it.

<Canvas>
  <Story name="ellipsis">
    <Layout>
      <Page>

      </Page>
    </Layout>
  </Story>
</Canvas>
